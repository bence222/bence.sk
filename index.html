O M N E

Programujem v HTML, CSS a Pythone.Tieto tri jazyky tvoria základ moderného webového vývoja a s ich pomocou je možné vytvárať dynamické, interaktívne a vizuálne atraktívne webové stránky a aplikácie.

HTML (Hypertext Markup Language) je základným jazykom pre tvorbu webových stránok. Pomocou HTML definujeme štruktúru a obsah stránky.

CSS (Cascading Style Sheets) je jazyk, ktorý sa používa na definovanie vzhľadu a štýlu webových stránok vytvorených v HTML. Pomocou CSS môžeme meniť farby, písma, veľkosti a rozloženie elementov na stránke.

Python je vysokoúrovňový interpretovaný programovací jazyk, ktorý sa používa na širokú škálu aplikácií.

PHP (Hypertext Preprocessor) je skriptovací jazyk, ktorý je špecificky navrhnutý na vývoj webových aplikácií. PHP umožňuje vytvárať dynamické webové stránky, ktoré sa prispôsobujú interakcii s používateľom. Môže spracovávať dáta z formulárov, komunikovať s databázami a generovať dynamický obsah na základe konkrétnych podmienok.
